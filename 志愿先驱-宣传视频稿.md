# 启明志愿宣传视频稿（20秒）

## 配音文案

```
高考志愿不是只看分数。

真正拉开差距的，往往是信息差。

启明志愿把公开录取数据、专业就业方向和志愿填报方法放到一起，
帮你先看清楚：这个分数大概在哪个层次，
哪些专业适合，哪些方向要慎重。

你说清楚省份、位次、选科和目标，
它会先问关键问题，
再给你冲、稳、保几个参考方向。

不替你拍板，
但尽量帮你把选择看明白。

免费，开源。希望每个普通家庭都少走一点弯路。
```

| 秒数 | 画面 | 配音 |
