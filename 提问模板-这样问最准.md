# 这样问，Agent 回答最准

> 把下面模板里的【括号内容】换成你自己的，直接复制发过去。

---

## 模板一：高考生选专业（最常用·详细版）

> 👇 复制下面这段话，把【】里的内容全换成你自己的

```
【省份】考生，选科【物理/历史/物化生/史政地/物化地】。

分数：今年考了【分数】分
位次：全省排名大概【位次】名（没有就写不知道）

家里情况：爸妈是【做什么工作的】，家庭年收入大概【多少】，
亲戚朋友里有在【电力/铁路/医院/学校/银行/政府/没有】行业工作的吗。

想去哪儿：【武汉/省内/北上广/南方/离家近就行】，能不能接受去外省，
最远能接受【本省/邻省/全国都行/只要不是新疆西藏】。

学科偏好：比较擅长的科目是【数学/物理/英语/历史/都一般】，
最讨厌的科目是【数学/物理/背书/英语/都还好】，
绝对不能接受的专业方向是【学医/当老师/写代码/下工地/没有】。

未来打算：毕业以后想【找高薪工作/考公务员考编制/考研深造/稳定就行/还没想好】，
对工作有什么期待吗，比如【能接受加班/想955不加班/愿意去工厂/只想坐办公室】。

帮我详细分析适合什么专业，推荐冲稳保学校。
```

## 模板二：不知道学什么（帮你理方向）

```
我是【省份】的，今年高考，【物理/历史】类考了【分数】分。
我现在完全不知道学什么专业好，也不确定想去哪。
家庭情况是【简单描述】。
你先问我几个问题，帮我理一下方向。
```

## 模板三：有具体目标，让Agent评估

```
我想学【专业名称】，想去【学校或城市】。
我的情况是【省份】【物理/历史】类，考了【分数】分，排名【位次】名。
家里情况是【简单描述】。
你帮我看看这个方向合不合适？有什么坑？有没有更好的选择？
```

```

2. 【学校B】的【专业B】
3. 【学校C】的【专业C】

我是【省份】【物理/历史】类，【分数】分，排名【位次】。
家里【简单描述家庭情况】。
你帮我对比一下，哪个更适合我？为什么？
```


## 小技巧

1. **信息越全越准**：省份、分数、位次、家庭情况、讨厌什么、想要什么——六样说齐了，Agent 直接给方案，不用反复追问。

2. **位次比分数重要**：如果有全省排名（位次），一定带上。分数每年波动，位次才是硬通货。

3. **讨厌什么和喜欢什么一样重要**：说清楚不能接受什么（比如"绝对不学医""看到数学就头疼"），Agent 帮你避开。

4. **家里干什么的别不好意思说**：爸妈在电力、铁路、医院、学校、做生意的——这决定了很多隐藏的好路。

5. **多轮聊效果更好**：Agent 会追问你一些问题，认真回答。聊得越深，建议越准。
